🚀 Deploy Smart Return Furor to Railway

Follow these steps to permanently host your app so it works in Shopify Admin at all times.

Step 1 — Create a Shopify App in Partners
  1. Go to partners.shopify.com → Apps → Create app (Custom app)
  2. Name it Smart Return Furor
  3. Copy the API key and API secret
Step 2 — Deploy to Railway
  1. Go to railway.app → New Project → Deploy from GitHub repo
  2. Push the generated app folder to a GitHub repo and connect it
  3. Set these Environment Variables in Railway:
SHOPIFY_API_KEY=your_api_key
SHOPIFY_API_SECRET=your_api_secret
HOST=https://your-app.railway.app
SCOPES=read_orders,write_orders,read_products
NODE_ENV=production
  1. Railway will auto-detect npm start — make sure it's defined in package.json
  2. After deploy, note your Railway URL (e.g. https://smart-return-furor.railway.app)
Step 3 — Configure URLs in Shopify Partners
  1. App URL: https://your-app.railway.app/
  2. Redirect URL: https://your-app.railway.app/auth/callback
  3. Embedded app: Yes
  4. Required scopes: read_orders, write_orders, read_products
Step 4 — Install on Your Store

Visit: https://your-app.railway.app/auth?shop=your-store.myshopify.com

After OAuth, your app will load inside Shopify Admin automatically via App Bridge.
